# 01_rtk_reverse_nav_save_dir_baseline

## 定位

这是“添加保存目录版”的基础稳定版本，主要用于验证两点模式直线倒车和轨迹保存目录功能。它是后续停车、继电器、影子模式和喷洒控制版本的母版。

## 核心功能

- 使用 RTK/INS 位姿作为主定位来源。
- 支持两点确定直线后倒车。
- 支持路径记录、导入、导出。
- 支持 `path_dir`、`path_prefix` 等保存目录参数。
- 控制核心仍是连续跟踪：航向误差 + 横向误差小幅修正。

## 为什么保留

它代表“没有停车/继电器/喷洒机构干扰时，底盘能稳定倒车”的基线。后续版本出现问题时，都应回到这个版本理解：原本连续控制是可用的，后来的问题多来自停车状态机和上方喷洒机构耦合。

## 工程效果

适合做长直线倒车基线测试。该阶段的经验是：只要保持连续控制，不频繁打断导航，车辆可以较稳定地沿两点直线倒回。
